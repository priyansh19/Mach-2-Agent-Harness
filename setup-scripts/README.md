# Mach2 setup scripts

| Script | Purpose |
|--------|---------|
| [langfuse/setup.ps1](langfuse/setup.ps1) | Self-host Langfuse (Windows) → writes `mach2-config.env` |
| [langfuse/setup.sh](langfuse/setup.sh) | Same for macOS/Linux |

Run Langfuse setup first, then copy `mach2-config.env` → `.env` at repo root.
